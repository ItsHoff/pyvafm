Detailed Description

Active Band Pass Filter circuit.

schema

Takes a signal in and passes it through a Band pass filter using the Sallen-Key topology

Initialisation parameters:

  • gain = Integer How much gain the signal will recive
  • fc = the frequency cut off for the circuit
  • band = The band of frequncies that will be filtered.
  • pushed = True|False push the output buffer immediately if True

Input channels:

  • signal = incoming signal

Output channels:

  • out = where is the input signal, is the input signal, is the cut off pulse is the quality factor and is the timestep.

Examples:

1 machine.AddCircuit(type='SKBP', name='filter', fc=50, band=5, pushed='True')
2 machine.AddCircuit(type='SKBP', name='filter', gain=10, fc=50, band=5, pushed='True')
