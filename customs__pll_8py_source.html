10 ## \brief Analog PLL composite circuit.
11 # Assembly function for analog PLL composite circuit.
12 # The Phase-Frequency Detector (PFD) multiplies the signals, and the result
13 # is passed to a series of lowpass Sallen-Key filters. The mount of filters
14 # and their cutoff frequencies are taken from the input parameter \a filters.
15 #
16 # \note For this to work, the output \a sin should be connected to \a signal2 in the parent circuit (as shown in example).
17 # Also, both signals should be normalised (amplitude = 1) and their offset should be removed.
18 #
19 #
20 # \htmlonly <div class="image"><a href="apll.png"><image src="apll.png" alt="aPLL schema" /></a> \endhtmlonly
21 #
22 # \b Initialisation \b parameters:
23 # - filters = list of cutoff frequencies for PFD lowpass filters
24 # - Kp = proportional constant of the charge pump
25 # - Ki = integral constant of the charge pump
26 # - gain = gain on the charge pump output
27 # - pushed = True|False push the output buffer immediately if True
28 #
29 # \b Input \b channels:
30 # - \a signal1 = incoming signal
31 # - \a signal2 = reference signal
32 # - \a f0 = fundamental frequency
33 #
34 # \b Output \b channels:
35 # - \a sin = sine wave of the internal VCO
36 # - \a cos = cosine wave of the internal VCO
37 # - \a df = frequency shift from \a f0
38 #
39 #
40 # \b Example:
41 # \code{.py}
